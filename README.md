# SolidStart

## Setup

First run

```bash
bun install 
```

Then setup db
```bash
bun run generate 
bun run migrate 
bun run push 
```

Now you can run dev server

### or start the server and open the app in a new browser tab
```bash
bun run dev -- --open
```

## Build the app
```bash
bun run build 
```
